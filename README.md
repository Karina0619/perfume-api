Что внутри репозитория:
server.js - сам код на Node.js (с Express)
package.json и package-lock.json - это для установки библиотек
Также есть скриншоты всех запросов: регистрация, логин, добавление товара, список товаров, заказ, отзывы
Были использованы методы: GET и POST.
Что работает:
1) Регистрация и вход по токену
2) Админ может добавлять товары, обычный пользователь - нет
3) Ограничение на запросы: максимум 100 в час
4) Можно создать заказ и оставить отзыв
Статус-коды ошибок:
400 - не заполнены обязательные поля
401 - требуется авторизация
403 - доступ запрещён
404 - ресурс не найден
409 - email уже существует
429 - слишком много запросов
Стратегия сопровождения API:
Версионирование (/api/v1/),каждая новая версия API будет иметь свой номер в адресе
Обратная совместимость - 6 месяцев
Публичный Changelog (список изменений), все что будет меняться, буду записывать в отдельный файл
